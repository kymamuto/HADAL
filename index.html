// STEP 01 — 게임 소개
HADAL이란?
2183년, 기후변화로 대부분의 대륙이 바닷속에 잠겼습니다.

인류의 생존을 위해 당신은 심해 채굴 작전을 수행합니다. 심해 바닥에 매장된 수소(H2)를 채굴해 수송선의 연료로 사용하는 것이 목표입니다.

하지만 방사능에 오염된 기형 괴생물체들이 당신의 채굴을 방해합니다.
🎯 최종 목표
15턴 안에 최대한 많은 수소를 채굴하세요.
채굴한 수소량이 다음 수송전의 이동력이 됩니다.
✓ 채굴 성공 조건
모든 H2 거점을 채굴하면 즉시 성공 — 이동이 가능해도 종료됩니다.

// STEP 03 — 기본 조작
어떻게 움직이나요?
초록 테두리가 표시된 칸을 탭하면 해당 칸을 점령합니다. 내 영역에 인접한 칸만 점령할 수 있습니다.
H2 거점 채굴 방법
1. H2 거점을 탭해 점령
2. 3턴 동안 유지 (빼앗기지 않게!)
3. 3턴 후 자동으로 채굴 완료 → 수소 획득
⚠ 주의
H2 거점 점령 후 괴생물체에 빼앗기면 카운터가 초기화됩니다. 주변을 먼저 확보하세요.
TURN 소모 규칙
탭 1회 = 1턴 소모. SKIP TURN 버튼으로 턴만 넘길 수도 있습니다.
// STEP 04 — 위협 : 괴생물체
☢ 괴생물체를 조심하세요
괴생물체는 매 턴 인접 칸을 점령하며 영역을 확장합니다. 내 영역과 H2 거점을 우선적으로 노립니다.
☢ 확장 속도
초반(1~7턴) : 매 턴 1칸 확장
중반 이후 : 매 턴 2칸 확장

추가 군락이 5턴, 10턴, 14턴에 맵 가장자리에서 등장합니다.
⚡ SURGE — 돌진
매 5턴(5·10·15턴)에 발동. 선두 1마리가 기지 방향으로 최대 3칸 돌진합니다.
발동 1턴 전 경고 배너가 뜨니 BARRIER로 경로를 차단하세요.
🧫 SPORE — 포자 분열
초반 1~6턴 사이 랜덤 2회 발동. 플레이어 영역 근처(2~4칸)에 새 군락이 기습 출현합니다.
DRONE으로 즉시 해당 위치를 선점하거나 BARRIER로 차단하세요.
종료 조건
모든 인접 칸이 괴생물체에 막히면 이동 불가 — 채굴 종료로 게임이 끝납니다.
// STEP 05 — 스킬 시스템
BARRIER & DRONE
각 스킬은 한 판에 2회만 사용 가능합니다. 타이밍이 핵심입니다.
⬡ BARRIER — 장벽 설치
사용법: BARRIER 버튼 탭 → 노란 테두리 칸 탭
설치 가능: 내 타일 또는 내 타일에 인접한 빈 칸 (1칸 이내)
효과: 해당 칸이 장벽이 되어 괴생물체가 진입 불가
제한: H2 거점, 기지, 채굴 완료 타일, 괴생물체 영역 불가
~ SONAR LURE — 괴생물체 유인
사용법: SONAR 버튼 탭 → 청록 테두리 칸(3칸 내) 탭
효과: 선두 괴생물체 1마리가 선택한 칸으로 이동
비용: 수소 -1 (수소가 0이면 사용 불가)
횟수: 제한 없음 — 수소만 있으면 계속 사용 가능
예외: 괴생물체에 빼앗긴 H2 거점으로는 유인 가능
사용법: DRONE 버튼 탭 → 보라 테두리 칸 탭
효과: 최대 7칸 거리의 빈 칸을 즉시 점령 (1턴 소모)
제한: H2 거점, 괴생물체 영역에는 사용 불가
// STEP 06 — 전략 가이드
15턴을 어떻게 써야 할까?
거점이 기지에서 멀수록 수소량이 많지만 시간도 많이 걸립니다. 어떤 거점을 포기할지 결정하는 것도 전략입니다.
초반 (1~5턴)
빠르게 영역 확장 → 가능한 많은 H2 거점 점령 → 괴생물체와 거리 벌리기
중반 (6~10턴)
점령한 H2 거점 사수 → BARRIER로 핵심 경로 차단 → 추가 군락 등장 주의
후반 (11~15턴)
DRONE으로 고수익 거점 선점 → 포위 전 마지막 채굴 완료 집중
